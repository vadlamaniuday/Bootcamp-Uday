Notes - Weather Widget

LLMs Used : No

Abstract

In this app, I had to create a weather app where I had to use external API from OpenWeatherMap API to get the data.


The handicap on this was, we have to use Vanilla JS on this. This was an easy task given that we were allowed to use Vite for building and such.

Approach

The approach I took was :

  1. Generate an API Key.
  2. Display the data.

Resources I used

A lot of resources (mainly for config) came in handy here including but not limited to :

  1. OpenWeatherMap
  2. StackOverflow
