Abstract

In this exercise, I had to create a capstone project where I had to retrieve 1000 tables of data, and on that data, I had to perform filtering operations. The handicap on this was, we have to use Vanilla JS/TS on this. This was an easy task given that we were allowed to use Vite for building and such.

Approach

The approach I took was :

  1. Getting the Data
    1. I got the data from a kaggle dataset (link)
    2. I cleaned the csv file from a Python (Specifically pandas) script which takes the top 1000 rows in the dataset and converts and stores the converted data inside a JSON file.
  2. I then used the data for building the functionality and the UI around it.

Resources I used

A lot of resources (mainly for config) came in handy here including but not limited to :

  1. LLMs
  2. Kaggle

Improvements

I believe that there's a room for improvement around debouncing and the time taken to load the table.
