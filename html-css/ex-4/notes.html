Exercise 4: CSS Units and Responsive Design

Goal: Practice using various CSS units and make the design responsive.

Introduction: Learn to work with different CSS units and responsive techniques. Know about px, em, rem, ch, etc. Also units like vh, vw. Learn about min, max, and clamp() to help in creating scalable designs.

Level 1:

Create a centered webpage layout using %, vh, and vw. Center a content block horizontally and vertically.

Level 2:

Use min(), max(), and clamp() for font sizes and widths. Style the page content to scale proportionally across different screen sizes.

Level 3:

Implement continuous responsiveness by avoiding fixed breakpoints. Use advanced responsive techniques like flexible grids and fluid typography.

  1. Solution 1

    Findings
    • px (pixels) are useful for precise control but not ideal for responsive design.
    • em are useful for scaling elements relative to their parent's font size.
    • rem are relative to the font-size of the root element and provide a consistent scaling across the entire document.
    • % are useful for creating fluid layouts because they are relative to the parent element's size.
    • vh (viewport height) and vw (viewport width) are used for creating elements that scale with the viewport height and width respectively.

  2. Solution 2

    Findings
    • min and max return the smallest and largest value among the provided values respectively.
    • clamp() "clamps" a value between a minimum and maximum value and is useful for creating responsive typography and layouts.

  3. Solution 3

    Findings
    • Responsiveness ensures that the webpage adapts dynamically across all screen sizes and devices by avoiding fixed breakpoints.
    • Fluid typography means that text sizes adjust proportionally to the viewport dimensions to maintain aesthetic consistency and readability.
