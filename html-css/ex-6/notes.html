Goal: Following the best resources available

Do the following steps:

  1. Set up daily.dev as your homepage. Customize your feed. You will see all the relevant recent articles.
  2. Subscribe to JavaScript Weekly.
  3. Subscribe to CSS Weekly.
  4. Subscribe to This Week in React.
  5. Read annual issues of:
    • State of JavaScript
    • State of CSS
    • State of Frontend
  6. Subscribe to 5 people (or organizations) on YouTube from the ecosystem (usually mentioned in the annual surveys).
