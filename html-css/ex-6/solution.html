Learnings

  1. State of JS
    • TypeScript continues to gain traction and its adoption is driven by improved developer experience and strong support from major frameworks like React and Angular
    • New meta-frameworks like Next.js and Nuxt.js are enhancing development experiences across various frameworks
    • React, Angular, and Vue.js remain the leading frameworks, but newer contenders like Svelte and SolidJS are gaining popularity
  2. State of CSS
    • Developers reported significant issues with cross-browser compatibility, particularly with features like anchor positioning and the Popover API.
    • The survey indicated a growing interest in utility-first frameworks like Tailwind CSS
    • New CSS logo, with the color of the logo called as Rebecca Purple
  3. State of Frontend
    • SPAs are dominating over multi-page applications due to their ability to load content dynamically without full page reloads.
    • The JAMstack architecture(JavaScript,API,Markup) is gaining traction, allowing developers to create static HTML pages served via CDNs
