Exercise 2: CSS Variables and Basic Styling

Goal: Learn to use CSS variables and apply basic styles to an HTML page.

Introduction: Understand the cascading nature of styles. How do you override a style? How do you customize a style? What ways can you theme a style? Check out light and dark modes. How CSS variables help manage consistent styling across a webpage. Explore how to define and use variables for colors, fonts, and spacings.

Level 1:

Define five CSS variables in variables.css:

  • Primary and secondary colors.
  • Font size for headers and body.
  • Spacing for margins.
Link the variables file to your main CSS file and use them to style the page from Exercise 1.

Level 2:

Use default values for variables when they are not set (e.g., var(--primary-color, black)). Style alternate rows of the table using CSS selectors (e.g., nth-child).

Level 3:

Use variables to define other variables (e.g., --secondary-color: var(--primary-color);). Create a dynamic theme switcher using variables to toggle between light and dark modes.
