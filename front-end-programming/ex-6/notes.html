FRONT-END PROGRAMMING - Exercise 6

Notes

I created a basic color picker website by using Vanilla JavaScript with Vite where users can adjust the background and table colors by picking colors and adjusting the HSL Sliders.

HTML Structure

HTML is structured in an article type of way along with few more changes with respect to addition of tables and sliders

Javascript Structure

  • Adding event listeners to the color pickers and sliders to detect changes.
  • Updating the corresponding HSL values and applying them to the background and table elements.
  • Ensuring that the updates are smooth and do not cause performance issues.
